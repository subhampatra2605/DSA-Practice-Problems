# ✅ **Problem 11: Two Sum II – Input Array Is Sorted**

## 🔹 **Problem Statement**

Given a **sorted** array, return **1-indexed** indices of two numbers that add up to the target.

---

## 📥 Input

numbers = [2,7,11,15], target = 9

## 📤 Output

[1,2]

---

# 💀 Brute Force

Check each pair using two loops.

**TC:** O(n²)
**SC:** O(1)

---

# 🚀 Optimal (Two Pointers)**

Because the array is sorted:

* if sum < target → move left pointer forward
* if sum > target → move right pointer backward
* if sum == target → return positions

**TC:** O(n)
**SC:** O(1)

---

# ✅ Java Code (Optimal)

```java
class Solution {
    public int[] twoSum(int[] numbers, int target) {
        int l = 0, r = numbers.length - 1;

        while (l < r) {
            int sum = numbers[l] + numbers[r];

            if (sum == target) return new int[]{l + 1, r + 1};
            else if (sum < target) l++;
            else r--;
        }

        return new int[]{};
    }
}
