# ✅ **Problem 7: Valid Sudoku**

## 🔹 **Problem Statement**

Determine if a given **9×9 Sudoku board** is valid based on the rules:

* No duplicate digits in any row
* No duplicate digits in any column
* No duplicate digits in any 3×3 box

(Empty cells are represented by '.')

---

## 📥 Example Input (Partial)

```
[
["5","3",".",".","7",".",".",".","."],
["6",".",".","1","9","5",".",".","."],
...
]
```

## 📤 Expected Output

```
true / false
```

---

# 💀 Brute Force

Check every row, column, and 3×3 box separately with nested loops.

**TC:** O(9×9×9)
**SC:** O(1)

(Not preferred.)

---

# 🚀 Optimal (HashSet Tracking)

Track seen values using encoded strings:

* `"5 in row 0"`
* `"5 in col 1"`
* `"5 in box 0-0"`

If a value repeats in any category → invalid.

**TC:** O(81)
**SC:** O(81)

---

# ✅ Java Code (Optimal)

```java
class Solution {
    public boolean isValidSudoku(char[][] board) {
        HashSet<String> seen = new HashSet<>();

        for (int r = 0; r < 9; r++) {
            for (int c = 0; c < 9; c++) {
                char val = board[r][c];
                if (val == '.') continue;

                if (!seen.add(val + " in row " + r)) return false;
                if (!seen.add(val + " in col " + c)) return false;
                if (!seen.add(val + " in box " + (r/3) + "-" + (c/3))) return false;
            }
        }
        return true;
    }
}
