# 📌 120. Triangle

---

# 🧠 Problem Type

* **Category:** Dynamic Programming
* **Pattern:** Bottom-Up DP (Minimum Path Sum)
* **Concept:** Each cell depends on next row’s adjacent values

---

# 🎯 Core Idea

From position `(i, j)` you can move to:

```text
(i + 1, j)
(i + 1, j + 1)
```

So:

```text
dp[i][j] = triangle[i][j] + 
            min(dp[i+1][j], dp[i+1][j+1])
```

---

# 🚀 Best Approach → Bottom-Up (Optimal)

Instead of going top-down,
start from **second last row** and move upward.

Why?
Because bottom row is already the base case.

---

# 💻 Optimal Java Solution (O(n) Space)

```java
import java.util.*;

class Solution {
    public int minimumTotal(List<List<Integer>> triangle) {
        int n = triangle.size();
        
        // Copy last row into dp array
        int[] dp = new int[n];
        for (int i = 0; i < n; i++) {
            dp[i] = triangle.get(n - 1).get(i);
        }
        
        // Start from second last row
        for (int row = n - 2; row >= 0; row--) {
            for (int col = 0; col <= row; col++) {
                dp[col] = triangle.get(row).get(col) + 
                          Math.min(dp[col], dp[col + 1]);
            }
        }
        
        return dp[0];
    }
}
```

# 🧪 Example Walkthrough

Triangle:

```
   2
  3 4
 6 5 7
4 1 8 3
```

Start with bottom row:

```
dp = [4,1,8,3]
```

Move upward:

Row 2:

```
6 + min(4,1) = 7
5 + min(1,8) = 6
7 + min(8,3) = 10
dp = [7,6,10,3]
```

Row 1:

```
3 + min(7,6) = 9
4 + min(6,10) = 10
dp = [9,10,10,3]
```

Row 0:

```
2 + min(9,10) = 11
```

Answer = 11

# ⏱ Complexity

* **Time:** O(n²)
* **Space:** O(n)

Where n = number of rows

---

# 🏆 Why Bottom-Up Is Better

✔ No recursion
✔ No extra 2D DP table
✔ Space optimized
✔ Interview preferred

---


# 🎯 Interview Explanation (Short)

> Starting from the bottom row, I compute the minimum path sum upward. Each cell becomes its value plus the minimum of the two adjacent cells below. Finally, dp[0] gives the answer.
