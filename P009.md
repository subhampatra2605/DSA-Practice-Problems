# ðŸš€ Optimal (HashSet + Only Start From Sequence Start)**

Steps:

1. Insert all numbers in a HashSet.
2. For each number, check if it's the **start** of a sequence
   (`n-1` must NOT be in set).
3. Count how long the sequence goes.

**TC:** O(n)
**SC:** O(n)

# âœ… Java Code (Optimal)

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        HashSet<Integer> set = new HashSet<>();
        for (int n : nums) set.add(n);

        int longest = 0;

        for (int n : set) {
            if (!set.contains(n - 1)) {       // start of sequence
                int curr = n;
                int streak = 1;

                while (set.contains(curr + 1)) {
                    curr++;
                    streak++;
                }

                longest = Math.max(longest, streak);
            }
        }

        return longest;
    }
}
