# 📌 796. Rotate String

---

# 🧠 Problem Type

* **Category:** String
* **Pattern:** String Rotation / Trick
* **Concept:** Concatenation + Substring check

---

# 🎯 Core Goal

Given two strings `s` and `goal`:

👉 Return `true` if `s` can become `goal` after **any number of left rotations**

---

# 🔄 What is Rotation?

```text
"abcde" → "bcdea" → "cdeab" → "deabc" → "eabcd"
```

---

# 🚀 Key Insight (VERY IMPORTANT)

Instead of rotating manually:

👉 A rotated string will always be a **substring of (s + s)**

---

# 💡 Why This Works

If:

```text id="p9c8pu"
s = abcde
```

Then:

```text id="6l9mb7"
s + s = abcdeabcde
```

All rotations:

```text id="v9v7y0"
bcdea, cdeab, deabc, eabcd
```

👉 All exist inside `s + s`

---

# ❗ Important Condition

Before checking:

```text id="i3h3pz"
length(s) == length(goal)
```

If not → return false immediately

---

# 🧩 Strategy

1. Check length
2. Create `s + s`
3. Check:

```text id="bt8h7z"
(s + s).contains(goal)
```

---

# 💻 Optimal Java Code

```java id="nlqdfz"
class Solution {
    public boolean rotateString(String s, String goal) {

        if (s.length() != goal.length()) return false;

        String doubled = s + s;

        return doubled.contains(goal);
    }
}
```

---

```

```
