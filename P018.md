# 📌 9. Palindrome Number

---

# 🧠 Problem Type

* **Category:** Math / Number Manipulation
* **Pattern:** Half Reversal
* **Goal:** Check palindrome without converting to String

---

# 🚫 Important Observations

1. If `x < 0` → ❌ Not palindrome
   (Negative sign makes it invalid)

2. If `x % 10 == 0` and `x != 0` → ❌ Not palindrome
   Example:

   ```
   10 → reversed becomes 01
   ```

---

# 🚀 Optimal Approach (Half Reversal)

Instead of reversing entire number (risk overflow),
we reverse only **half** of the digits.

---

## 🔁 Core Logic

While:

```text
x > reversedHalf
```

Keep extracting last digit and build reversedHalf.

At the end:

* If even digits → `x == reversedHalf`
* If odd digits → `x == reversedHalf / 10`

---

# 💻 Optimal Java Code

```java
class Solution {
    public boolean isPalindrome(int x) {
        
        // Negative numbers are not palindrome
        if (x < 0 || (x % 10 == 0 && x != 0)) {
            return false;
        }

        int reversedHalf = 0;

        while (x > reversedHalf) {
            reversedHalf = reversedHalf * 10 + x % 10;
            x /= 10;
        }

        // For even digits: x == reversedHalf
        // For odd digits: ignore middle digit (reversedHalf / 10)
        return x == reversedHalf || x == reversedHalf / 10;
    }
}
```

---

# 🧪 Example Walkthrough

### Input:

```
121
```

Steps:

```
reversedHalf = 1, x = 12
reversedHalf = 12, x = 1
```

Stop when `x <= reversedHalf`.

Check:

```
x == reversedHalf / 10
1 == 12 / 10 → 1 == 1 ✔
```

---

# ⏱ Complexity

* **Time:** O(log10(n))
* **Space:** O(1)

We process half the digits only.

---

# 🏆 Why This Is Optimal

✔ No string conversion
✔ No full reversal
✔ No overflow risk
✔ Single pass over digits

---

# 🎯 Interview Explanation (Short Version)

> I reject negative numbers and trailing zero cases. Then I reverse only half of the number to avoid overflow. Finally, I compare the first half with the reversed half.

---

# 🔥 Key Takeaways

* Use `% 10` to extract digit
* Use `/ 10` to remove digit
* Reverse only half for safety
* Handle edge cases early
